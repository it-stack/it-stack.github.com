使用Hexo建立个人博客

  1. Hexo的主要命令
    http://hexo.io/docs/commands.html

  2. Hexo建站的主要参考地方
    http://ibruce.info/2013/11/22/hexo-your-blog/

  3. 一些优化技巧和markdown语法
    http://baoxiehao.com/2014/05/17/Hexo%E5%8D%9A%E5%AE%A2%E4%BC%98%E5%8C%96/
    http://ibruce.info/2013/11/26/markdown/

  4. 我采用的主题
    http://hahack.tk/hexo-theme-freemind/

  5. 快速命令
    hexo new “创建第一篇博文”
    hexo server //启动本地hexo服务器
    hexo generate //生成本地静态文件

  6. FAQ
    1) 如何删除一篇文章呢？
    删除相应的markdown文件，和博客本地根目录的db.json文件，重新generate就可以了。
